## PDF Generator

The `generate_professional_pdfs.py` script generates beautifully formatted PDF manuals from the text files using ReportLab.

### Requirements

```bash
pip install reportlab
```

### Usage

```bash
python3 generate_professional_pdfs.py
```

This will generate all PDF manuals with:
- Professional cover pages
- Consistent branding and styling
- Headers and footers with page numbers
- Properly formatted warning and note boxes
- Smart page breaks
